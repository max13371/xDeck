# 📦 TrackPack

**Система отслеживания посылок на Swift**

## 🚀 Возможности
- 🔍 Отслеживание посылок
- 🔔 Push-уведомления
- 📍 Геолокация
- 📱 Нативный iOS интерфейс

## 🛠 Технологии
- **iOS:** Swift, SwiftUI
- **Backend:** Vapor (Swift)
- **База данных:** PostgreSQL
- **Уведомления:** Apple Push Notification Service
